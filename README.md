Для роботи програми необхідне наступне програмне забезпечення: операційна система Windows, компілятор g++ (MinGW) версії 4.8 або вище. Також програма використовує системну функцію `Beep()` та `Sleep()`, які доступні лише на платформі Windows.

Компіляція та запуск виконується певною командою у терміналі g++ Main.cpp Sound.cpp -o Main.cpp

Після запуску програма автоматично створює два треки та додає їх до плейлиста. Керування програмою здійснюється через інтерактивне текстове меню. Опис пунктів меню наведено у таблиці 3.1.

**Таблиця 3.1 – Опис пунктів меню**

| Пункт | Функція | Опис |
|---|---|---|
| `1 - show` | `showAll()` | Виводить список усіх треків з індексами та кількістю нот |
| `2 - listen` | `play()` | Відтворює обраний трек через динамік |
| `3 - status` | `printStatus()` | Виводить назву, темп, тональність та кількість нот треку |
| `4 - tonalnist` | `shiftTone()` | Зсуває тональність треку на задану кількість позицій від -6 до +6 |
| `5 - revers` | `reverseTrack()` | Реверсує порядок нот обраного треку |
| `6 - zlyyty` | `mergeTrack()` | Об'єднує два обраних треки в один новий |
| `7 - vydalyty` | `removeTrack()` | Видаляє обраний трек з плейлиста |
| `0 - vyhid` | — | Завершує роботу програми |

Для вибору треку програма запитує його індекс зі списку. Для пункту `4 - tonalnist` додатково вводиться значення зсуву у діапазоні від -6 до +6. Для пункту `6 - zlyyty` вводяться індекси двох треків, які необхідно об'єднати.
